# 연산자?
값에 대해 어떤 작업을 컴퓨터에 지시하기 위한 기호
```javascript
a = 1   // 대입 연산자
```

# 비교 연산자
주어진 값들이 같은지, 다른지, 큰지, 작은지 구분한다.  
비교 연산자의 결과는 true나 false다.  
true와 false는 boolean이라 불리는 데이터 형식이고, 이는 조건문에서 자세히 다룬다.
