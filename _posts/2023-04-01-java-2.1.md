# 수의 표현
```html
<!DOCTYPE html>
<html>
<head>
  <title></title>
</head>
<body>
<script type="text/javascript">
  alert(1);
</script>
</body>
</html>
```
위의 코드를 입력하면 1이라는 숫자로 팝업창이 뜨는것을 확인할 수 있다.  
JavaScript에서는 정수, 실수라는 개념이 나뉘어져 있다기 보단, 숫자라는 개념으로 통일된다.  
연산자는 그대로 사용할 수 있다.  
`alert(1.1+1);` → '2.1'  
`alert(2*8);` → '16'  
`alert(6/2);` → '3'  

# 수의 연산
```html
Math.pow(3,2)
```
`pow` 명령어를 이용해 '제곱'을 구현할 수 있다.
```html
Math.round(10.6) 
Math.ceil(10.6)
Math.floor(10.6)
```
```html
Math.sqrt(9)
```
```html
Math.random()
```
```html
Math.round(100*Math.random())
```
