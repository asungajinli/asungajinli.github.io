  1. fnmatch
    파일 이름을 패턴으로 매칭시키기 위한 함수 제공.
    함수
    • fnmatch.fnmatch(name, pat) : name이 pat과 매칭되면 True
    • fnmatch.fnmatchcase(name, pat) : name이 pat과 매칭되면 True, 대소문자 구분
    • fnmatch.filter(names, pat) : names에서 pat과 매칭되는 이름만 반환
    • fnmatch.translate(pat) : 패턴을 정규식으로 변환
    패턴
    • * : 모든 문자열
    • ? : 한 문자
    • [seq] : seq 안에 있는 문자 중 하나
    • [!seq] : seq 안에 없는 문자 중 하나
    예제
    • fnmatch.fnmatch('foo.txt', '*.txt') : True
    • fnmatch.fnmatch('foo.txt', '?oo.txt') : True
    • fnmatch.fnmatch('Dat45.csv', 'Dat[0-9]*') : True
    • fnmatch.fnmatch('foo.txt', '*.TXT') : False
  2. os
    운영체제와 상호작용하기 위한 함수 제공.
    파일 및 디렉토리 조작, 환경 변수 액세스, 프로세스 관리 등 다양한 작업을 수행.
    함수
    • os.name : 운영체제 이름
    • os.environ : 환경 변수
    • os.getenv(key[, default]) : 환경 변수 값
    • os.putenv(key, value) : 환경 변수 설정
    • os.unsetenv(key) : 환경 변수 삭제
    • os.getcwd() : 현재 작업 디렉토리
    • os.chdir(path) : 작업 디렉토리 변경
    • os.system(command) : 시스템 명령어 실행
    • os.listdir(path) : 디렉토리 내 파일 목록
    • os.remove(path) : 파일 삭제
    • os.rename(src, dst) : 파일 이름 변경
    • os.mkdir(path) : 디렉토리 생성
    • os.rmdir(path) : 디렉토리 삭제
    • os.path.join(path, *paths) : 경로 생성
    • os.path.abspath(path) : 절대 경로
    • os.path.relpath(path) : 상대 경로
    • os.path.dirname(path) : 디렉토리 이름
    • os.path.basename(path) : 파일 이름
    • os.path.exists(path) : 파일 존재 여부
    • os.path.isfile(path) : 파일 여부
    • os.path.isdir(path) : 디렉토리 여부
    • os.path.getsize(path) : 파일 크기
    • os.path.split(path) : 디렉토리와 파일 이름
    • os.path.splitext(path) : 파일 이름과 확장자
    • os.path.expanduser(path) : 사용자 디렉토리 치환
    • os.path.commonprefix(list) : 공통 경로
    • os.path.normpath(path) : 경로 정규화
    • os.path.samefile(path1, path2) : 동일 파일 여부
    • os.walk(top[, topdown=True[, onerror=None[, followlinks=False]]]) : 디렉토리 트리 순회
    • os.path.walk(top, topdown=True, onerror=None, followlinks=False) : 디렉토리 트리 순회
    • os.path.expandvars(path) : 환경 변수 치환

    외부 프로세스를 생성하고 제어하는 기능을 제공.
    다른 프로그램을 실행하고 그 결과를 가져오는 등의 작업을 수행할 때 사용.

    • subprocess.check_output(args, *, stdin=None, stderr=None, shell=False, universal_newlines=False) : 외부 명령어 실행, 성공 시 결과 반환
    • subprocess.Popen(args, bufsize=-1, executable=None, stdin=None, stdout=None, stderr=None, preexec_fn=None, close_fds=True, shell=False, cwd=None, env=None, universal_newlines=False, startupinfo=None, creationflags=0, restore_signals=True, start_new_session=False, pass_fds=(), *, encoding=None, errors=None) : 외부 명령어 실행, Popen 객체 반환

  8. threading
    스레드 기반의 병렬 프로그래밍을 지원하는 기능을 제공.
    동시에 여러 작업을 처리하거나 동기화된 작업을 수행할 때 사용.
    함수
    • threading.Thread() : 스레드 생성
    • threading.activeCount() : 활성화된 스레드 수
    • threading.currentThread() : 현재 스레드 객체
    • threading.enumerate() : 활성화된 스레드 목록
    • threading.activeCount() : 활성화된 스레드 수
    • threading.Lock() : 락 객체 생성
    • threading.RLock() : 재진입 락 객체 생성
    • threading.Condition() : 조건 객체 생성
    • threading.Semaphore() : 세마포어 객체 생성
    • threading.BoundedSemaphore() : 바운드 세마포어 객체 생성
    • threading.Timer() : 타이머 객체 생성
    • threading.Event() : 이벤트 객체 생성
    • threading.Barrier() : 바리어 객체 생성
    • threading.Thread.isAlive() : 스레드가 활성화되었는지 확인
    • threading.Thread.getName() : 스레드 이름
    • threading.Thread.setName(name) : 스레드 이름 설정
    • threading.Thread.isDaemon() : 데몬 스레드인지 확인
    • threading.Thread.setDaemon(daemonic) : 데몬 스레드 설정
    • threading.Thread.start() : 스레드 시작
    • threading.Thread.run() : 스레드 실행
    • threading.Thread.join(timeout=None) : 스레드 종료 대기
