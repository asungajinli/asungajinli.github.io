  1. LogLevel
    로그 레벨을 지정하는 열거형.
    Class & Level
    • LOG_NONE
      로그를 출력하지 않음.
    • LOG_ERROR
      오류 발생 시 로그를 출력.
    • LOG_WARN
      경고 발생 시 로그를 출력.
    • LOG_INFO
      정보 발생 시 로그를 출력.
    • LOG_FUNCTION
      함수 호출 시 로그를 출력.
    • LOG_LOGIC
      논리적 오류 발생 시 로그를 출력.
    • LOG_DEBUG
      디버깅 시 로그를 출력.
    • LOG_ALL
      모든 로그를 출력.
    More
  2. LogComponent
    로그 컴포넌트를 나타내는 클래스.
    LogComponentEnable
    로그 컴포넌트를 활성화하는 함수.
    LogComponentEnable("LogComponent", 0); // 로그 컴포넌트를 활성화
    LogComponentDisable
    로그 컴포넌트를 비활성화하는 함수.
    LogComponentDisable("LogComponent", 0); // 로그 컴포넌트를 비활성화
    LogComponentPrintList
    로그 컴포넌트의 목록을 출력하는 함수.
    LogComponentPrintList("LogComponent", 0); // 로그 컴포넌트의 목록을 출력
    LogComponentPrintAll
    모든 로그 컴포넌트의 목록을 출력하는 함수.
    LogComponentPrintAll(); // 모든 로그 컴포넌트의 목록을 출력

  3. CommandLine
    명령행 인자를 처리하는 클래스.
    AddValue
    명령행 인자를 추가하는 함수.
    CommandLine cmd; // 명령행 인자를 처리하는 객체
cmd.AddValue("arg", "arg description", 0); // 명령행 인자를 추가
    Parse
    명령행 인자를 파싱하는 함수.
    cmd.Parse(argc, argv); // 명령행 인자를 파싱
    GetNCommandLine
    명령행 인자의 개수를 반환하는 함수.
    int n = cmd.GetNCommandLine(); // 명령행 인자의 개수를 반환
    GetCommandLine
    명령행 인자를 반환하는 함수.
    char *arg = cmd.GetCommandLine(0); // 명령행 인자를 반환

  7. PointToPointHelper
    Point to Point 네트워크를 생성하는 Helper.
    SetDeviceAttribute
    네트워크 장치의 속성을 설정하는 함수.
    PointToPointHelper p2p; // Point to Point 네트워크 생성 Helper
p2p.SetDeviceAttribute("DataRate", DataRateValue(5 * 1000 * 1000)); // 데이터 전송 속도 설정
    SetChannelAttribute
    채널의 속성을 설정하는 함수.
    PointToPointHelper p2p; // Point to Point 네트워크 생성 Helper
p2p.SetChannelAttribute("Delay", TimeValue(MilliSeconds(2))); // 전송 지연 시간 설정
    Install
    노드에 네트워크 장치를 설치하는 함수.
    NodeContainer n; // 노드 컨테이너 생성
NetDeviceContainer d; // 네트워크 장치 컨테이너 생성
d = p2p.Install(n); // 노드에 네트워크 장치를 설치

  9. Ipv4AddressHelper
    IPv4 주소를 생성하는 Helper.
    SetBase
    네트워크 주소와 넷마스크를 설정하는 함수.
    같은 주소가 두 번 생성되는 실수를 발생시키면 치명적인 오류가 발생한다 (이는 매우 디버그하기 어려운 오류이므로 유의해야한다).
    Ipv4AddressHelper ipv4; // IPv4 주소 생성 Helper
ipv4.SetBase("10.1.1.0", "255.255.255.0"); // 네트워크 주소와 넷마스크 설정
    Assign
    노드에 IPv4 주소를 할당하는 함수.
    NodeContainer n; // 노드 컨테이너 생성
NetDeviceContainer d; // 네트워크 장치 컨테이너 생성
Ipv4InterfaceContainer i; // IPv4 인터페이스 컨테이너 생성
i = ipv4.Assign(d); // 노드에 IPv4 주소를 할당
    More
  10. UdpEchoServerHelper

    노드에 UDP 에코 서버를 설치하는 함수.
    NodeContainer n; // 노드 컨테이너 생성
ApplicationContainer a; // 애플리케이션 컨테이너 생성
a = echoServer.Install(n.Get(1)); // 노드에 UDP 에코 서버를 설치

  13. CsmaHelper
    CSMA/CD 링크를 생성하는 Helper.

    링크의 속도를 설정하는 함수.
    CsmaHelper csma; // CSMA/CD 링크 생성

d = csma.Install(n); // 노드에 CSMA/CD 링크를 설치
