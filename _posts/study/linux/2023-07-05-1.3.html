  1. ls
    디렉토리의 내용 확인
    ls [옵션] <디렉토리>

    ls -al
ls -al test_dir1
  2. cd
    디렉토리 변경
    cd <디렉토리>

    cd test_dir1
  3. pwd
    현재 디렉토리 확인
    pwd
  4. mkdir
    디렉토리 생성
    mkdir [옵션] <디렉토리>

    mkdir test_dir1
mkdir -p test_dir1/test_dir2
  5. rmdir
    디렉토리 삭제
     rmdir [옵션] <디렉토리>

     rmdir test_dir1
  6. rm
    파일 또는 디렉토리 삭제
     rm [옵션] <파일>

     rm test1.txt
 rm -r test_dir1
  7. cp
    파일 또는 디렉토리 복사
    cp [옵션] <복사할 파일> <복사될 파일>

    cp test1.txt test_dir2/
cp -r test_dir1 test_dir2
  8. mv
    파일 이동
    mv [옵션] <이동할 파일> <이동될 파일>

    mv test_dir1/test1.txt test_dir2/
mv test1.txt test2.txt
  9. cat
    파일 내용을 출력
    cat <파일>

    cat test1.txt
  10. head
    파일의 앞부분 출력
     head [옵션] <파일>
    • -n
      n줄만 출력
       head -n 3 test1.txt
  11. grep
    파일에서 문자열 검색
     grep [옵션] <문자열> <파일>
    • -i
      대소문자 구분하지 않고 검색
       grep -i "hello" test1.txt
    • -v
      검색하지 않는 문자열 출력
       grep -v "hello" test1.txt
    • -n
      줄 번호 출력
       grep -n "hello" test1.txt
    • -r
      하위 디렉토리까지 검색
       grep -r "hello" test_dir1
    12. no results
