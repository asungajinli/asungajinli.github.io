# 주석 comment
- 한줄 주석
  ```javascript
  // 한줄 주석
  ```
- 여러줄 주석
  ```javascript
  /*
    여러줄 주석
    여러줄 주석
    여러줄 주석
    여러줄 주석
  */
  ```

# 줄바꿈과 여백
- 세미콜론(;)
  - 명령이 끝났다는 의미
  - 줄이 바뀔 때 반드시 써야 하는것은 아님
  - 줄이 바뀌지 않을 땐 반드시 입력
- `tab`, 띄어쓰기
  - 가독성을 높임
