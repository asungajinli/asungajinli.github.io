# 사용법
JavaScript에서 변수는 var로 시작한다.  
var은 생략 가능하지만 이는 유효범위라는 것에 영향을 미친다.  
유효범위에 대해서는 뒤에서 다룬다.
```javascript
var a = 1;
alert(a);             // 1

a = 10;
alert(a);             // 10
```
```javascript
a = 1;
b = 2;

alert(a+b);           // 3
```
